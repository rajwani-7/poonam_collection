# 📸 Image Guidelines - Poonam Collection

## Image Requirements

### Product Images
- **Size:** 1000x1000px (minimum 800x800px)
- **Format:** JPG, PNG, WebP
- **Max File Size:** 5MB per image
- **Background:** Plain white or light background preferred
- **Quality:** High resolution, clear and sharp

### Category Images
- **Size:** 800x600px
- **Format:** JPG
- **Max File Size:** 3MB
- **Style:** Lifestyle or product showcase

### Hero/Banner Images
- **Size:** 1920x1080px
- **Format:** JPG
- **Max File Size:** 5MB
- **Style:** Eye-catching, brand representative

## Image Optimization Tips

### Before Upload
1. Resize to recommended dimensions
2. Compress without quality loss
3. Use descriptive file names
4. Remove EXIF data

### Tools for Optimization
- **Online:** TinyPNG, ImageOptim
- **Software:** Photoshop, GIMP

## Photography Tips

### DIY Product Photography

**Equipment Needed:**
- Smartphone camera (8MP+)
- Natural lighting or ring light
- White backdrop (cloth/paper)
- Tripod or stable surface

**Steps:**
1. Place product on white background
2. Use natural light from window
3. Take from multiple angles
4. Keep camera steady
5. Edit brightness/contrast if needed

**Angles to Capture:**
- Front view
- Back view
- Side view
- Close-up of details
- Product in use (model wearing)

## Image Naming Convention

Use descriptive names for better organization:

```

Examples:
- silk-saree-red-front-1.jpg
- designer-kurti-blue-back-2.jpg
- bridal-lehenga-gold-closeup-3.jpg
```

## Image SEO Tips

### File Names
- Use hyphens, not underscores
- Include product name and color
- Keep it descriptive but short

### ALT Text (In HTML)
```html
<img src="silk-saree-red.jpg" alt="Red Silk Saree with Golden Border">
```

## Troubleshooting

### "Image too large"
- Compress image using TinyPNG
- Resize to recommended dimensions
- Check file size < 5MB

### "Upload failed"
- Check internet connection
- Verify uploads folder permissions
- Check PHP upload limits in php.ini

### "Image not displaying"
- Clear browser cache
- Check file path is correct
- Verify image format is supported

## Best Practices

✅ **Do:**
- Use consistent lighting
- Keep backgrounds simple
- Show product details clearly
- Upload multiple angles
- Optimize before upload
- Use descriptive names

❌ **Don't:**
- Use blurry images
- Mix different backgrounds
- Upload oversized files
- Use copyrighted images
- Forget to compress
- Use generic file names
